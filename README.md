# Bangor River Level App

Mobile-first river monitoring app for EPA station 33008.

## Features

- Current river level estimate
- EPA level (lagged)
- Gauge conversion
- Trend detection (rising/falling/steady)
- Forecast (1h / 3h / 6h)
- Interactive chart (6h–3d views)
- EPA reference image
- Offline support (PWA)

## Tech

- Vanilla JavaScript
- Chart.js
- GitHub Pages
- Service Worker caching

## Data Source

EPA Hydronet station:
https://epawebapp.epa.ie/hydronet/

## Deployment

Enable GitHub Pages:
Settings → Pages → Deploy from branch → main / root
